Zapier Running Route Bot

An AI-powered Telegram bot for personalized running routes

Project Overview

The Zapier Running Route Bot is a Telegram bot that generates personalized running routes based on user inputs like distance, run type, and mood. It integrates real-time weather data and provides motivational messages and Google Maps links for navigation.

Built using Zapier, Telegram API, OpenWeatherMap API, and custom JavaScript, this project showcases automation, API integration, and user-friendly design.

Key Features

  • Parses natural language requests (e.g., "5km easy run")
  • Fetches real-time weather data via OpenWeatherMap API
  • Generates circular running routes with Google Maps links
  • Provides motivational messages tailored to weather and run type
  • Handles multiple run types (easy, tempo, interval, recovery)

Code Samples

Below is a snippet of the JavaScript code used to parse user inputs in Zapier:

const message = inputData.message.toLowerCase();
let distance = 5;
let unit = "km";
let runType = "easy";
let mood = "neutral";
const distanceMatch = message.match(/(\d+(?:\.\d+)?)\s*(km|k|miles?|mi|m)/i);
if (distanceMatch) {
  distance = parseFloat(distanceMatch[1]);
  unit = distanceMatch[2].toLowerCase();
  if (unit.includes('m') && !unit.includes('km')) {
    unit = "miles";
  } else {
    unit = "km";
  }
}
if (message.includes('tempo')) runType = 'tempo';
if (message.includes('interval')) runType = 'interval';
if (message.includes('recovery')) runType = 'recovery';
if (message.includes('long')) runType = 'long';
if (message.includes('easy')) runType = 'easy';
output = [{ distance, unit, runType, mood, originalMessage: message }];
